우커머스에서 다음 결제방식을 사용할 수 있습니다.
- 나이스페이 (신용카드, 실시간 계좌이체, 가상계좌이체, 휴대폰 결제)
- 나이페이 빌링 키 (정기결제)
- 카카오페이

== Description ==
대한민국에서 사용 가능한 우커머스 결제용 PG 플러그인을 찾고 계신가요?
이 플러그인은 나이스페이 API를 사용하여 신용카드, 가상계좌, 실시간 계좌, 휴대폰 결제를 지원합니다.
그리고 정기 결제를 위한 나이페이 빌링 API 또한 지원합니다.

이제 쇼플릭 우커머스 결제 플러그인에서 카카오페이를 사용할 수 있습니다.


== Installation ==

= Minimum Requirements =
* PHP 7.4 이상을 필요로 합니다.
